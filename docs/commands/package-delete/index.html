package-delete

Delete a package version from bintray.

SYNOPSIS

rerun bintray:package-delete --user <> --apikey <> --org <> --package <> --repo <>

OPTIONS

  • --user <>: The bintray subject, aka username.
  • --apikey <>: The password used for REST authentication.
  • --org <>: The bintray organization.
  • --package <>: The package name.
  • --repo <>: The targeted repo.

README

Deletes a package and any versions and files. If the package does not exist, the command returns sucessfully.

See bintray:package-exists to check if a package exists.
