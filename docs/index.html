bintray

Manage packages in bintray.

SYNOPSIS

rerun [ARGS] bintray: [ package-delete | package-exists | package-update | package-upload ] [OPTIONS]

README

The bintray commands help you manage packages in bintray.

Use package-upload to create a package version and upload a file.

Use package-delete to delete a package version from a repository.

To check if a package exists, use package-exists.

Use package-update to set the package description and labels.

Authentication

You must have a valid bintray login and API key.

COMMANDS

  • package-delete: Delete a package version from bintray.
  • package-exists: Checks if the package exists.
  • package-update: Update the information of the specified package.
  • package-upload: Uploads and publishes a package file.

OPTIONS

  • apikey: The password used for REST authentication.
  • description: The package description.
  • descriptor: File containing package metadata.
  • file: The file to upload and publish.
  • labels: The package labels.
  • org: The bintray organization.
  • package: The package name.
  • repo: The targeted repo.
  • user: The bintray subject, aka username.
  • version: The package version.
