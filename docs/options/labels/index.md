# labels

The package labels.

## SYNOPSIS

        --labels <>

## README

Values for the `--labels` option must be a comma separated list.

Here are some valid examples:

    --labels foo,bar,baz
    --labels foo
    
Do not use spaces or quotes. 

*Invalid:*

    --labels "foo","bbar","baz"
    --labels "foo, bar,   baz"
