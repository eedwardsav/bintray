bintray

apikey

The password used for REST authentication.

SYNOPSIS

    --apikey <>

README

The combination of a valid --user and --apikey are used to login to bintray. An API key can be obtained from the user profile page.

See the authentication section of the Bintray REST API.

COMMANDS

The following commands use this option.

  • package-delete
  • package-exists
  • package-update
  • package-upload
